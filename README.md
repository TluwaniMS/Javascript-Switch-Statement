# Javascript-Switch-Statement

The javascript `switch` statement is used to execute/trigger specific functions depending on the conditions that have been given.

```
const animal = "Cat";

switch (animal) {
  case "Cat":
    console.log(`Meow`);
    break;
  case "Dog":
    console.log(`Woof`);
    break;
  case "Cow":
    console.log(`Mooo`);
    break;
  default:
    console.log(`Woof Meow Mooo!!`);
    break;
}
```
